Technologies

Além das principais ferramentas (HTML, CSS, JS) tenho experimentado o React por conta da facilidade que ele oferece para integrar um mesmo componente à várias páginas diferentes, tornando bem prático a manutenção das páginas. Pensando na rapidez com que consigo tornar uma página responsiva,gosto do Bootstrap, que agregado ao flexbox facilita bastante tarefas deste tipo.

Como ferramenta de versionamento de código trabalho com Git, administrando e implementando modificações em repositórios do Github. Na hora de desenvolver no Backend, tenho utilizado NodeJS e APIs Restful. No momento, como tenho desenvolvido mais a parte de front, de sites e sistemas simples, utilizo geralmente MongoDb como Banco de Dados. Mas também possuo conhecimento com banco relacional My SQL Server, Oracle e Postgres.
