About

Sou um desenvolvedor frontend, focado em web e com experiência nas principais ferramentas de desenvolvimento para front: HTML, CSS e JavaScript. Gosto de utilizar o Bootstrap e o React, como frameworks em meus projetos. O React por conta da facilidade que ele oferece para integrar os componentes da interface com várias páginas diferentes… e o Bootsrap pensando na experiência do usuário, de forma responsiva.

Como ferramenta de versionamento de código trabalho com Git, administrando e implementando modificações em repositórios do Github. Na hora de linkar as minhas interfaces com o banco de dados, coloco em cena o Axios para fazer requisições APIs Rest no Backend, onde, geralmente, uso NodeJS.Como Bancos de Dados relacional faço uso de My SQL Server 2016. Já como não relacional, MongoDB.

Sou focado, paciente, resiliente e busco sempre trazer a inovação e a criatividade para o meu dia a dia. Honestidade, responsabilidade e determinação são valores importantes para mim. Tenho uma verdadeira paixão por estudar idiomas e conhecer novas culturas.
